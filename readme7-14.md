## Steps to push code to Github

- Create folder for code 
`mkdir nameOfFolder`
- Navigate into newly created folder 
`cd nameOfFolder`
- Create Local Git  Repository
`git init`
- Create code
`touch readme.md`
- Track / Stage / Watch


```
    git add .
```

 or

```
  git add -A
```
- Commit our changes to our local git 
```
    git commit -m "First commit"
```
-  Create a github repo
